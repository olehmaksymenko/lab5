In order to improve road safety, lower maintenance costs, and enhance the driving experience overall, this project intends to create a complex system that can monitor and analyze road surface conditions.  
Fundamentally, the system makes use of information gathered from car sensors to recognize and evaluate different aspects of the road surface, including unevenness, cracks, and potholes.

An agent module, a primary data processing module, a data accumulation module, a storage component, and a visualization module are among the essential elements of the system's architecture, which is in line with earlier stages of development.  
Together, these elements collect, process, and display information about the state of the roads.

Laboratory work №4 is currently centered on the application of "edge data logic." 
 This vital component of the system consists of:

 * On-the-spot road condition analysis; * 
 Real-time data collection through MQTT;  
 Analyzed data is efficiently forwarded to a central "Hub."

 The system will be able to give quick feedback on road conditions by putting this edge data logic into practice, enabling prompt maintenance and increased road safety.  
 Essentially, the goal of this project is to develop a proactive, intelligent system that helps make road infrastructure safer and more effective.
